# v1.0.0 — UUIDv7 minimal

Première version stable, minimale mais utilisable en production.

- **Statut** : en cours de finalisation (code et qualité OK, reste CI + tag + Packagist)
- **Cible** : génération, parsing et validation d'UUIDv7 conformes RFC 9562

## Scope

Strictement limité à la génération, au parsing et à la validation d'UUIDv7.
Aucune autre fonctionnalité ne doit être ajoutée dans cette version.

## API publique

| Méthode | Description |
|---------|-------------|
| `Uuid7::generate(): self` | Génère un UUIDv7 (timestamp ms + aléa `random_bytes`). |
| `Uuid7::fromString(string $uuid): self` | Parse une forme canonique ; lève `InvalidUuidException` si invalide. |
| `Uuid7::toString(): string` | Forme canonique 8-4-4-4-12, normalisée en minuscules. |
| `Uuid7::__toString(): string` | Idem via `Stringable`. |

### Exceptions

| Exception | Levée quand |
|-----------|-------------|
| `Exception\InvalidUuidException` | `fromString()` reçoit une chaîne non conforme (longueur, hex, version ≠ 7, variant hors RFC, tirets manquants). |

## Détails techniques

- Disposition 128 bits conforme RFC 9562 : 48 bits timestamp (big-endian),
  4 bits version (`0b0111`), 12 bits aléa, 2 bits variant (`0b10`), 62 bits aléa.
- Source de vérité interne : 16 octets bruts (prépare `toBytes`/`toHex` en v1.3).
- Aléa exclusivement `random_bytes()` (jamais `rand`/`mt_rand`/`uniqid`).
- `fromString()` est insensible à la casse ; `toString()` renvoie toujours du minuscule.
- Value object `final`, immuable (constructeur privé), implémente `Stringable`.
